## هيكل المشروع :

# 🧠 نظام  العرب (Arab OS)

هذا المشروع هو نظام تشغيل (Operating System) تم تطويره من الصفر باستخدام لغة C و Assembly، مع هيكلية واضحة تشمل الإقلاع، النواة، ومساحة المستخدم (لايزال تجريبي يحتاج تطوير).

---

## 📁 هيكل المشروع

### 🔹 boot
مسؤول عن عملية الإقلاع:
- `grub/` → إعدادات GRUB
- `multiboot.asm` → كود بدء تشغيل النواة
- `default.png` → صورة الإقلاع
- `الخطوط/` → خطوط العرض

---

### 🔹 أرض_المستخدم (User Space)
تمثل بيئة المستخدم داخل النظام:
- `النظام/` → التطبيقات والخدمات
- `المستخدمين/` → إدارة المستخدمين
- `الواجهة/` → الطرفية (Terminal)
- `البيانات/` → بيانات النظام
- `التكوين/` → إعدادات النظام
- `الذاكرة_المؤقتة/` → الكاش
- `سجلات/` → ملفات السجلات

---

### 🔹 المثبت (Installer)
خاص بتثبيت النظام:
- `الواجهة/` → واجهة المستخدم للمثبت
- `موارد/` → ملفات وموارد التثبيت

---

### 🔹 النواة (Kernel)
قلب النظام والمسؤول عن إدارة العتاد والعمليات:

#### 📌 التعريفات الأساسية (Drivers)
- التخزين (ATA)
- الشاشة (VGA)
- لوحة المفاتيح
- نظام ملفات مخصص (`نظام_العرب`)

#### 📌 القلب (Core)
- `الجوهرة.c` → منطق أساسي للنواة

#### 📌 المحركات (Engines)
- `محرك_الذاكرة` → إدارة الذاكرة
- `محرك_العمليات` → إدارة العمليات
- `محرك_الجدولة` → جدولة المهام
- `محرك_التنفيذ` → تنفيذ البرامج
- `محرك_الفيديو` → الرسوميات
- `محرك_النصوص` → معالجة النصوص
- `محرك_أنظمة_الملفات` → إدارة الملفات

---

### 🔹 ملفات أخرى
- `Makefile` → أوامر البناء
- `ربط_الملفات.ld` →  ربط الملفات البرمجية
- `تفاصيل/` → توثيق إضافي

---
